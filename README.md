# Git Exercicios

Projeto simples em HTML criado para praticar versionamento com Git e organizacao de paginas estaticas com navegacao entre arquivos.

## Visao geral

Este repositorio funciona como um exercicio de estrutura basica para web e Git. A pagina principal faz a navegacao para paginas dedicadas a tecnologias e frameworks estudados.

## Conteudo do projeto

- `index.html`: pagina principal
- `react.html`: pagina sobre React
- `vue.html`: pagina sobre Vue
- `angular.html`: pagina sobre Angular
- `spring.html`: pagina sobre Spring Boot

## Tecnologias

- HTML5

## Como visualizar

Basta abrir o arquivo abaixo no navegador:

```text
index.html
```

Tambem e possivel usar a extensao Live Server no VS Code para navegar entre as paginas localmente.

## Objetivos de aprendizado

- praticar criacao de arquivos HTML interligados
- entender navegacao com links
- organizar um projeto simples para versionamento
- exercitar fluxo de `add`, `commit` e `push`
